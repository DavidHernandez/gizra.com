# Gizra
> The gizra.com website

## Install

```bash
bundler install
npm install
bower install
```

## Grunt tasks

Serve locally

``grunt serve``

Serve locally a from the built ``dist`` directory (no livereload)

``grunt serve:dist``

Deploy to gizra.com

``grunt deploy``
