Our Products

What is Openscholar
OpenScholar is a solution to the Babel Tower problem many higher learning establishments face.
Rather than having multiple department, staff and project websites written in a multitude of environments, OpenScholar provides a single, open source codebase for easily creating and maintaining academic websites, based on the field-proven Drupal CMS.

What is Garment Box
GarmentBox is a toolbox for small and medium fashion houses, providing an intuitive user interface for creating catalogs, managing collections, tracking stock, sales, customer relations etc.
What is Zariz
Zariz is a set of Drupal extensions that brings the world of continuous deployment and version control into content management, as well as allow easy exporting of Drupal sites into static content, ready to be hosted on GitHub Pages or highly performant other static page hosts.
